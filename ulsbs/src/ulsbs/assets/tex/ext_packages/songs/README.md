This directory contains the needed files from the `songs` package:

   - [songs.sty](songs.sty): the songs package, a required file for compiling
     the songbook
   - [songidx.lua](songidx.lua): used in creating indices, a required file for
     compiling the songbook

The full songs package distribution is found [here](../../../../../../misc/ext_package_songs_distribution/).
It contains documentation and samples. The files above are extracted from that
package.
